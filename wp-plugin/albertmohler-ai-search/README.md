# albertmohler-ai-search - WordPress Plugin

WordPress plugin for Albert Mohler YouTube semantic search.

## Setup

The plugin files are generated by running `08_build_wp_plugin_v1.py` after completing the server setup.

## Installation

1. Run `08_build_wp_plugin_v1.py` to generate plugin files
2. Zip the `albertmohler-ai-search` folder
3. Upload to WordPress via Plugins → Add New → Upload Plugin
4. Activate the plugin
5. Go to Settings → Albert Mohler Search to configure:
   - API URL (your server endpoint)
   - API Key

## Features (after generation)

- `[albertmohler_search]` shortcode for embedding search
- Video thumbnail display
- AI-powered summaries
- Click tracking analytics
- Admin settings page

## Files (generated by 08_build_wp_plugin_v1.py)

- `albertmohler-ai-search.php` - Main plugin file
- `includes/class-admin.php` - Admin settings page
- `includes/class-shortcode.php` - Search shortcode
- `includes/class-analytics.php` - Click tracking
- `public/css/search.css` - Frontend styles
- `public/js/search.js` - Frontend JavaScript
