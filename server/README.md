# AlbertMohler - Flask Search Server

This directory contains the Flask API server for semantic search.

## Setup

The server is generated by running `07_local_POC_v1.py` after completing the data pipeline.

## Local Development

```bash
# From project root
source venv/bin/activate
cd server
python app.py
```

Server will be available at: http://localhost:5007

## Endpoints

| Route | Method | Auth | Description |
|-------|--------|------|-------------|
| `/` | GET | None | Search interface |
| `/api/v1/health` | GET | Optional | Health check |
| `/api/v1/search` | POST | Optional | Search transcripts |
| `/api/v1/summarize` | POST | Optional | AI summary |
| `/api/v1/stats` | GET | Optional | Index statistics |
| `/admin` | GET | Basic Auth | Admin dashboard |

## Production Deployment

See `deploy/` directory for:

## Files

- `app.py` - Main Flask application (generated by 07_local_POC_v1.py)
- `static/index.html` - Search interface (generated by 07_local_POC_v1.py)
- `data/analytics.db` - SQLite analytics database (created at runtime)
- `requirements.txt` - Server-only dependencies (for production deployment)
